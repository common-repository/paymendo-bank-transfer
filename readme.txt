Accept payment by bank transfer on your WooCommerce store. Also easily manage payments made by bank transfer.

== Description ==
Define your bank accounts and share them with your customers. Moreover, increase the experience with e-mail and SMS notifications.

- You can add a bank account for all banks in Turkey.
- When your customers choose the transfer option, they will see your account numbers.
- They report I made the payment to the "that" account.
- You check your account and confirm the payment.
- With your approval, order payment is completed and other processes are active.

= Configuration =
- In the plugin, you can define your bank accounts that you will use.
- By going to the plugin settings, you can set the notifications and the content of the notifications.

= SMS Configuration =
SMS sending is done with the ["WP SMS Functions"](https://wordpress.org/plugins/wp-sms-functions/) plugin. To enable SMS sending, first install the plugin. All SMS sending settings will be made from there. (Visit the Settings / SMS Provider menu)

== Installation ==

This section describes how to install and run the plugin.

1. Upload the plugin files to the `/wp-content/plugins/paymendo-bank-transfer-lite` directory or install the plugin directly from WordPress plugins screen,
2. Activate the plugin from the 'Plugins' screen in WordPress,
3. Use the "shipmendo - Lite" screen to configure the plugin,
4. Select your favorite shipment provider(cargo company) and fill required settings,
